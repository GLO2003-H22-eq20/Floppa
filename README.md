# Floppa

Le meilleur site de vente anonyme au Québec!

## Requis

- Java 11
- Maven

## Setup

### Compilation

```
mvn clean install
```

### Exécution

```
mvn exec:java
```

### Checkstyle
```
mvn checkstyle:check
```
